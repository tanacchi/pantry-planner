# バックアップ・リストア運用ガイド

このドキュメントは、pantry-plannerプロジェクトのDB・設定・重要データのバックアップ・リストア運用方針をまとめたものです。

## 1. バックアップ方針

- DB・設定ファイルは定期自動バックアップ
- バックアップ先は安全なストレージ（暗号化）
- バックアップ世代・保持期間を明確化

## 2. リストア手順

- 障害・データ消失時は即時リストア
- リストア手順・検証は定期的に実施
- リストア後は整合性・動作確認

## 3. 運用ルール

- バックアップ/リストア手順はdocs/配下に記録
- 変更時は必ず手順も更新
