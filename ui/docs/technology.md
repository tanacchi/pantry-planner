# 使用技術（ui）

- 言語: TypeScript
- フレームワーク: Remix, React
- UI: Tailwind CSS
- 状態管理: React Context, useState, useEffect
- API連携: OpenAPI Generator, fetch, 独自APIクライアント
- テスト: Jest, Testing Library
- CI/CD: Netlify, GitHub Actions
- その他: liff, @line/liff, mkcert など
